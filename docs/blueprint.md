# **App Name**: PolitiTricks Exposed

## Core Features:

- Landing Page with Title and Subheading: Visually appealing landing page featuring the title "How to fool common people? - Political Edition" and a subheading "What this ebook is about" with the provided content.
- Payment Modal: A modal box pops up with a payment option button, initially set at 99 Rupees.
- Social Sharing Discount: Implement a discount system: clicking the sharing button enables a price drop from 99 to 13 Rupees upon sharing the website link with one person. It also ensures that a sharing link appears when users clicks this button.
- Price Drop Animation: A subtle animation is used to show the price dropping from 99 to 13 rupees after the user shares the link.
- Razorpay Integration: Integrate Razorpay payment gateway to process payments. Redirect to the payment page upon confirmation.
- Google Drive Redirection: After successful payment, redirect the user to the Google Drive link (https://drive.google.com/file/d/17yAepMItiG1JChoYNIvtMEejXEPA3IoB/view?usp=sharing) for ebook access.
- Access Google Drive: Add "Note* After payment, you can reacess the pdf file in your google drive" on the landing page

## Style Guidelines:

- Primary color: Saturated purple (#9400D3), drawing inspiration from the themes of political theater.
- Background color: Very light lavender (#F0F8FF), of similar hue to the purple, creating a high-contrast dark theme that's gentle on the eyes.
- Accent color: A contrasting light orange (#FF8C00) to highlight key interactive elements like buttons and the sharing option.
- Headline font: 'Playfair', serif with an elegant, high-end feel.
- Body font: 'PT Sans', sans-serif that combines a modern look with approachability, working well with Playfair's body text.
- Use flat, minimalist icons related to social sharing, payment methods, and document access. Preferably use satire theme icons to relate to political ebook.
- Single-page layout with a clear hierarchy, featuring the title and subtitle prominently, followed by the ebook description, payment options, and social sharing incentives.
- Subtle transition animations and engaging price-drop animation upon sharing.
